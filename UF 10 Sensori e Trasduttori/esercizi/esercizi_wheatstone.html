A2
Calcolo della tensione di squilibrio
Facile

R1 = R2 = R3 = 4700 ΩRx = 5100 ΩVcc = 5 V

  • Il ponte è bilanciato o sbilanciato? Giustifica numericamente.

R1 = R2 = R3 = R = 10000 ΩVcc = 5 V

Sezione C — Codice Arduino

3 esercizi
C1
Leggi e spiega il codice
Facile

Leggi attentamente il seguente sketch Arduino per la lettura di un ponte di Wheatstone e rispondi alle domande.

const float VCC = 5.0;
const float R1  = 10000.0;
const float R2  = 10000.0;
const float R3  = 10000.0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int rawB = analogRead(A0);
  int rawC = analogRead(A1);

  float V_B = (rawB / 1023.0) * VCC;
  float V_C = (rawC / 1023.0) * VCC;
  float Vout = V_B - V_C;

  float Rx = R2 * V_C / (VCC - V_C);

  Serial.print("Vout = "); Serial.print(Vout, 3);
  Serial.print(" V  |  Rx = "); Serial.println(Rx, 0);

  delay(500);
}
  • A cosa serve la riga float V_B = (rawB / 1023.0) * VCC? Perché si divide per 1023 e non per 1024?
  • Se rawC = 512, calcola a mano V_C e poi Rx. Cosa ti aspetti di vedere sul Serial Monitor?
  • Perché viene usato Serial.begin(9600) in setup() e non in loop()?
  • Cosa succederebbe se V_C fosse esattamente uguale a VCC (5,0 V)? Il codice darebbe un errore? Perché?
C2
Trova e correggi i bug
Medio

Il seguente codice contiene 4 errori (evidenziati con commenti rossi). Trovali, spiega il problema e riscrivi la riga corretta per ognuno.

const int VCC = 5;          // BUG 1
const float R1 = 10000.0;
const float R2 = 10000.0;
const float R3 = 10000.0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int rawB = analogRead(A0);
  int rawC = analogRead(A1);

  float V_B = (rawB / 1023) * VCC;   // BUG 2
  float V_C = (rawC / 1023.0) * VCC;

  float Rx = R2 * V_C / (VCC + V_C);  // BUG 3

  if (Rx = 10000) {                    // BUG 4
    Serial.println("Ponte bilanciato!");
  }

  Serial.print("Rx = "); Serial.println(Rx);
  delay(500);
}
  • Qual è il problema con const int VCC = 5? Come si corregge e perché è importante per i calcoli successivi?
  • Perché rawB / 1023 dà sempre 0? Come si corregge?
  • La formula di Rx usa VCC + V_C invece di VCC - V_C. Qual è il valore numerico errato se V_C = 2,5 V? E quello corretto?
  • Qual è la differenza tra = e == in C++? Riscrivi la condizione corretta per verificare il bilanciamento (considera una tolleranza di ±100 Ω).
C3
Scrivi il codice — media su più campioni
Medio

I segnali ADC di Arduino sono soggetti a rumore. Una tecnica semplice è fare la media di N letture consecutive prima di calcolare la tensione. Scrivi un programma che:

  • Legge N = 10 volte i pin A0 e A1 in rapida successione
  • Calcola la media dei 10 valori raw per ciascun pin
  • Converte le medie in tensione V_B e V_C
  • Calcola Vout e Rx
  • Stampa su Serial Monitor: media rawB, media rawC, V_B, V_C, Vout, Rx
  • Si ripete ogni secondo

Usa i seguenti parametri: R1 = R2 = R3 = 10 kΩ, Vcc = 5 V.

  • Scrivi il codice completo (puoi usare lo spazio sotto o un foglio separato).
  • Rispetto al codice senza media: come cambia il comportamento? Qual è il vantaggio?
  • Se N = 100 invece di 10, il programma funziona ancora bene? Ci sono rischi? (Suggerimento: pensa al tempo di esecuzione e all'overflow di variabili int.)

D4
Collega concetti — applicazione industriale
Medio

Un impianto industriale usa una cella di carico basata su un ponte di Wheatstone completo (4 strain gauge) per misurare il peso di materiale su un nastro trasportatore. L'uscita è nominalmente 2 mV/V (cioè 2 mV di Vout per ogni Volt di alimentazione al fondo scala di 100 kg). Il ponte è alimentato a Vcc = 10 V.

  • Calcola la tensione Vout massima (a 100 kg) e quella a metà scala (50 kg).
  • Puoi leggere questo segnale direttamente con l'ADC di Arduino (range 0–5 V, 10 bit)? Cosa ti servirebbe in più?
  • Spiega in breve perché si usa un ponte completo (4 elementi attivi) invece di un quarto di ponte: qual è il vantaggio in termini di sensibilità e compensazione della temperatura?
📐 Formulario di riferimento
Equilibrio: R1 × Rx = R2 × R3 → Rx = (R2 × R3) / R1
Nodo B: V_B = Vcc × R3 / (R1 + R3)
Nodo C: V_C = Vcc × Rx / (R2 + Rx)
Squilibrio: Vout = V_B − V_C
Rx da V_C: Rx = R2 × V_C / (Vcc − V_C)
Linearizzato: ΔVout ≈ (Vcc/4) × (ΔRx/R)    [ponte simmetrico, ΔRx << R]
ADC Arduino: V = (rawADC / 1023.0) × Vcc
